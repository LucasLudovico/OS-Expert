# OS-Expert
Sistema de Ordens de Serviço para Oficinas:

Este sistema foi concebido para centralizar e otimizar o fluxo de trabalho em oficinas mecânicas, garantindo o controle desde a entrada do veículo até a entrega final ao cliente. O objetivo principal é substituir processos manuais por uma gestão digital integrada que une estoque, mão de obra e prazos.

Proposta do Sistema
O sistema atua como o núcleo operacional da oficina, permitindo que diferentes perfis de colaboradores (atendentes, mecânicos e gestores) interajam em uma única plataforma para gerenciar as manutenções de forma organizada e transparente.

Funcionalidades Principais
Gestão de Usuários e Clientes
Autenticação de Mecânicos: Acesso seguro para cada profissional visualizar suas tarefas pendentes.

Cadastro de Clientes e Veículos: Registro detalhado de proprietários e histórico de manutenção por placa.

Controle de Insumos e Serviços
Gestão de Estoque: Cadastro e controle de peças com monitoramento de quantidades disponíveis.

Tabela de Preços: Definição padronizada de valores para peças e diferentes tipos de mão de obra.

Ciclo da Ordem de Serviço (OS)
Orçamentos Automatizados: Geração de orçamentos baseados na soma de peças necessárias e horas de serviço estimadas.

Distribuição Inteligente: Divisão de serviços entre mecânicos conforme suas especialidades técnicas (ex: motor, suspensão, elétrica).

Controle de Prazos: Monitoramento das datas de entrega prometidas ao cliente para evitar atrasos.

Regras de Negócio Implementadas
O início de qualquer serviço é condicionado à aprovação prévia do orçamento.

O sistema impede a atribuição de serviços de especialidades específicas para profissionais não qualificados na área.

Atualização de status em tempo real para permitir que a recepção informe o cliente sobre o progresso do reparo.
